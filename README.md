# 🛍️ Retail Sales Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** on a retail sales dataset to uncover customer purchasing patterns, sales trends, and business insights. Various visualizations and statistical techniques were used to understand the data and support data-driven decision-making.

---

## 📊 Dataset Information

- **Dataset:** Retail Sales Dataset
- **Records:** 1000+
- **Features:**
  - Transaction ID
  - Date
  - Customer ID
  - Gender
  - Age
  - Product Category
  - Quantity
  - Price per Unit
  - Total Amount

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📈 Exploratory Data Analysis

The following analyses were performed:

- ✅ Dataset Inspection
- ✅ Data Cleaning
- ✅ Missing Value Analysis
- ✅ Duplicate Value Check
- ✅ Statistical Summary
- ✅ Gender Distribution
- ✅ Age Distribution
- ✅ Monthly Sales Trend
- ✅ Product Category Analysis
- ✅ Average Purchase Amount by Gender
- ✅ Correlation Heatmap

---

## 📸 Visualizations

Project visualizations include:

- 📊 Monthly Sales Line Chart
- 👥 Gender Distribution
- 📦 Product Category Distribution
- 💰 Average Purchase Amount by Gender
- 🔥 Correlation Heatmap

---

## 💡 Business Insights

- Female and male customers have similar average purchase amounts.
- Monthly sales trends help identify high-performing months.
- Product category analysis highlights the most popular categories.
- Customer age distribution provides valuable demographic insights.
- Correlation analysis reveals relationships between numerical features.

---

## 🎯 Conclusion

The analysis provides valuable insights into customer behavior, purchasing trends, and sales performance. These findings can help businesses optimize inventory, improve marketing strategies, and enhance customer satisfaction through data-driven decisions.


5. Run all cells.
